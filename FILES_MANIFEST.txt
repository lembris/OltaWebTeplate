================================================================================
TROUBLESHOOTING GUIDE
================================================================================

Problem: Admin panel not accessible
Solution: Check admin login status, verify session

Problem: Videos not appearing on homepage
Solution: Check is_featured=1, verify is_active=1

Problem: Thumbnail not showing
Solution: Check video is public, verify video ID

Problem: Modal not opening
Solution: Check JavaScript console, clear cache

Problem: Database error
Solution: Run SQL migration, check table exists

Problem: Drag-drop not working
Solution: Check browser compatibility, verify SortableJS loaded

Problem: YouTube URL not recognized
Solution: Use full URL, check format, verify video exists

See YOUTUBE_VIDEOS_SETUP.md for detailed troubleshooting section.
