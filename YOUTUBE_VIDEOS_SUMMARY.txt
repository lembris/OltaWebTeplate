HOW TO USE
================================================================================

ADDING VIDEOS:
  1. Admin panel → YouTube Videos (from menu)
  2. Click "Add Video"
  3. Paste YouTube URL (auto-extracts video ID)
  4. Enter title, category, description
  5. Check "Active" and "Featured"
  6. Click "Add Video"
  7. Video appears on homepage automatically

EDITING VIDEOS:
  1. Dashboard → Find video → Click pencil icon
  2. Modify any field
  3. Click "Update Video"

DELETING VIDEOS:
  1. Dashboard → Find video → Click trash icon
  2. Confirm deletion

REORDERING VIDEOS:
  1. Dashboard → Drag by grip handle on left
  2. Drop in new position
  3. Auto-saves

TOGGLING STATUS:
  1. Dashboard → Use switch for "Active" status
  2. Use star icon for "Featured" status
  3. Changes save via AJAX (no page reload)

VIEWING ON WEBSITE:
  1. Homepage → Scroll to "Featured Video Content"
  2. 6 featured videos display in responsive grid
  3. Click play button → Modal opens
  4. Video autoplays
  5. Close: X button, click outside, or Escape key

================================================================================
TROUBLESHOOTING
================================================================================

VIDEOS NOT SHOWING:
  → Check is_featured = 1 in database
  → Check is_active = 1
  → Clear browser cache

YOUTUBE URL NOT WORKING:
  → Use full URL: https://www.youtube.com/watch?v=...
  → Video must be Public or Unlisted
  → Check video still exists on YouTube

THUMBNAIL NOT SHOWING:
  → Verify YouTube video is public
  → Check video ID is correct (11 characters)
  → Ensure YouTube thumbnail URL is accessible

ADMIN MENU NOT SHOWING:
  → Clear browser cache (Ctrl+F5)
  → Re-login to admin panel
  → Check Menu_helper.php was updated

DATABASE ERROR:
  → Run SQL migration from phpMyAdmin
  → Verify youtube_videos table exists
  → Check database credentials
